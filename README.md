## Proyecto

INVENTAR-IO es una aplicación web desarrollada para la gestión de inventario en pequeños negocios.

## Módulos integrados

- Registro de usuarios
- Inicio de sesión
- Registro de productos
- Consulta de productos
- Actualización de productos
- Eliminación de productos

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Maven
- Postman
- GitHub
